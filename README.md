# IT Support Practice Sites

Summary of the sites I'm using to practice IT support / ticketing, what each is for, and the steps to get set up on each. Notes, screenshots, etc. for each site go in its matching subfolder.

---

## 📁 ServiceNow
**What it's for:** Practicing with a real, enterprise-grade ITSM platform — the same tool most large companies actually use for incident management.

**Steps:**
1. Sign up for a free account at the [ServiceNow Developer Portal](https://developer.servicenow.com)
2. Request a Personal Developer Instance (PDI) — takes ~10–15 min to provision
3. Log in to the instance URL with the admin credentials emailed to you
4. Navigate to **Incident → All** to see preloaded demo tickets (PDIs come with sample data built in)
5. Practice: triage tickets, set priority, assign, add work notes, resolve, and close

## 📁 ServiceDesk Simulator
**What it's for:** Practicing day-to-day helpdesk *scenarios* — realistic tickets, live chat, AI-powered support calls, Active Directory tasks, and remote desktop troubleshooting. More gamified and beginner-friendly than ServiceNow, closer to actual on-the-job situations.

**Steps:**
1. Go to [servicedesk-simulator.com](https://servicedesk-simulator.com) on a desktop browser (JavaScript required)
2. Create a free account (no credit card needed for the free tier)
3. Start working through tickets — password resets, network outages, software installs
4. Try the AI voice calls, Active Directory tasks, and remote desktop tools as they unlock
5. Track progress via the leaderboard/achievements
6. (Optional) Upgrade to Pro ($20/mo) for unlimited tickets/calls if the free tier runs out
